# Deployment

The repo is connected to **Cloudflare Pages**. Just push to `main` — Cloudflare handles the build and deploy automatically.

## Steps

// turbo-all

1. Stage all changes:
```
git add -A
```

2. Commit with a descriptive message:
```
git commit -m "your commit message"
```

3. Push to `main`:
```
git push origin main
```
